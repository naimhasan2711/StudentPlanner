## Managing Tasks

### Creating a Task

1. Tap the **+ (Floating Action Button)** on any screen
2. Select **"Add Task"**
3. Fill in the task details:
   - **Title**: Brief description of the task
   - **Description**: Additional details (optional)
   - **Due Date**: When the task is due
   - **Priority**: Low, Medium, High, or Urgent
   - **Category**: Organize by subject or type
   - **Reminder**: Enable to get notified before the deadline

### Task Properties

#### Priority Levels
- üü¢ **Low**: Non-urgent tasks
- üü† **Medium**: Regular assignments
- üî¥ **High**: Important deadlines
- üî¥ **Urgent**: Critical, immediate attention needed

#### Categories
Organize tasks by:
- Assignment
- Exam
- Project
- Reading
- Custom categories

### Editing a Task
1. Tap on any task card
2. Modify the details
3. Save changes

### Completing a Task
Simply tap the **checkbox** on a task card to mark it as complete.

### Deleting a Task
1. Tap on a task to open details
2. Tap the **delete icon** (trash)
3. Confirm deletion

### Task Filters
In the Tasks screen, use tabs to filter:
- **All**: View all tasks
- **Incomplete**: Only pending tasks
- **Completed**: Finished tasks

### Bulk Actions
- **Delete All Completed**: Remove all completed tasks at once (available in Tasks screen)

## Managing Lectures

### Creating a Lecture

1. Tap the **+ (Floating Action Button)**
2. Select **"Add Lecture"**
3. Fill in the lecture details:
   - **Title**: Course name (e.g., "Data Structures")
   - **Instructor**: Professor's name
   - **Room**: Location (e.g., "Room 301")
   - **Day of Week**: Select the day
   - **Start Time**: When the lecture begins
   - **End Time**: When the lecture ends
   - **Color**: Choose a color for easy identification
   - **Recurring**: Toggle for weekly recurring lectures
   - **Notes**: Additional information

### Recurring vs One-Time Lectures

#### Recurring Lectures
- Automatically appear every week on the selected day
- Perfect for regular classes
- Example: "Monday 09:00 - 10:30"

#### One-Time Lectures
- For special sessions, guest lectures, or exams
- Set a specific date instead of a day of week
- Won't repeat

### Editing a Lecture
1. Tap on any lecture card
2. Update the information
3. Save changes

### Deleting a Lecture
1. Open the lecture details
2. Tap the delete button
3. Confirm deletion

## Notifications

### Enabling Task Reminders
When creating or editing a task:
1. Enable **"Reminder"** toggle
2. Set how many minutes before the due date to be notified
3. Save the task

### Enabling Lecture Reminders
When creating or editing a lecture:
1. Enable **"Reminder"** toggle
2. Set minutes before lecture starts (default: 15 minutes)
3. Save the lecture

### Notification Permissions
On Android 13+, you'll be asked to grant notification permission on first launch. This is required for reminders to work.

### Managing Notifications
- Notifications appear in your notification shade
- Tap a notification to open the app
- Swipe to dismiss

## Tips & Best Practices

### For Tasks

1. **Use Priorities Wisely**
   - Reserve "Urgent" for truly critical deadlines
   - Most tasks should be "Medium" or "High"

2. **Break Down Large Projects**
   - Create multiple smaller tasks instead of one large task
   - Track progress more effectively

3. **Set Early Reminders**
   - For important deadlines, set reminders 1-2 days in advance
   - Gives you time to prepare

4. **Use Categories**
   - Group related tasks together
   - Easier to find and organize

5. **Regular Review**
   - Check the Today screen each morning
   - Review incomplete tasks weekly

### For Lectures

1. **Color Coding**
   - Use different colors for different subjects
   - Quickly identify lectures at a glance

2. **Add Useful Notes**
   - Office hours
   - Prerequisites
   - Important dates

3. **Keep Room Info Updated**
   - Some classes may change rooms
   - Update immediately to avoid confusion

4. **Set Appropriate Reminders**
   - 15-30 minutes is usually sufficient
   - Adjust based on how far away the classroom is

### General Tips

1. **Daily Routine**
   - Start your day by checking the Today screen
   - Plan your day based on tasks and lectures

2. **Weekly Planning**
   - Review your calendar every Sunday
   - Add upcoming assignments as tasks

3. **Clean Up Regularly**
   - Delete completed tasks monthly
   - Remove old one-time lectures

4. **Backup Important Info**
   - Take screenshots of important schedules
   - Write down critical deadlines elsewhere too

5. **Theme for Time of Day**
   - Use light theme during day
   - Switch to dark theme at night

## Troubleshooting

### Notifications Not Working
1. Check app permissions in system settings
2. Ensure "Do Not Disturb" is not blocking notifications
3. Verify reminder is enabled for the task/lecture

### Data Not Saving
1. Ensure all required fields are filled
2. Check for error messages
3. Restart the app

### Theme Not Changing
1. Close and reopen the app
2. Check Settings screen to verify selection

### App Crashes
1. Update to the latest version
2. Clear app cache in system settings
3. Report the issue with details
